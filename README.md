# ðŸ•º Real-Time Human Pose Estimation using MediaPipe & OpenCV

This project implements real-time human pose estimation using MediaPipe Pose and OpenCV. It detects and tracks full-body landmarks (33 key points) from webcam input and visualizes them with skeletal connections.

# Features

    Real-time pose detection from webcam
    
    33-body landmark tracking
    
    Smooth skeleton rendering
    
    Auto-close window handling
    
    Lightweight & fast (MediaPipe + OpenCV)

# Technologies Used

    Python
    
    OpenCV
    
    MediaPipe (Pose Solution)


# Install dependencies
            pip install opencv-python mediapipe


# How to Run
             Pose_Estimation.py

Press Q to exit the program.
